O source code está no arquivo ./server.ts e o arquivo transpilado para JS em dist/server.js.

Necessária instalação do NodeJS.

Para rodar a aplicação, fazer o node rodar o arquivo transpilado. Ex: node dist/server.js 

Dentro do diretório /dist/public temos os arquivos do CLIENT-SIDE. No arquivo index.js é necessário alterar o ip de conexão para o ip da máquina que será usada como servidor.
